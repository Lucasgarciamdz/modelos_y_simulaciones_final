Modelos y Simulación

Unidad 1: Introducción al Espacio de Estados

Una guía de estudio para el alumno

Conceptos Fundamentales

Preguntas Clave de esta Sección:

  • ¿Cuál es la diferencia entre un modelo y una simulación?
  • ¿Por qué es preferible modelar antes de experimentar en la realidad?
  • ¿Cuáles son las etapas de un proyecto de simulación?

¿Qué es un Modelo?

Es una descripción analítica y abstracta de un sistema o proceso, formulada para estudiar su comportamiento.

Permite cuantificar las variaciones de sus parámetros y entender sus interrelaciones.

Hoy en día, se le conoce como Gemelo Digital (Digital Twin).

¿Por qué Modelar?

La experimentación directa sobre un sistema real puede ser inconveniente por razones de:

  • Seguridad: Ensayos destructivos, análisis de explosivos, etc.
  • Costo: Probar prototipos físicos es caro.
  • Practicidad: Estudiar sistemas en condiciones extremas (ej. temperatura en el corazón humano).

¿Qué es la Simulación?

Es el proceso de diseñar, desarrollar y experimentar con un modelo programable para analizar su comportamiento.

Es una técnica numérica computarizada que nos permite "jugar" con el modelo en diferentes escenarios.

Etapas de un Proceso de Simulación

  1. Definición del sistema y formulación del problema: ¿Qué queremos estudiar?
  2. Formulación del modelo: Construcción conceptual y matemática.
  3. Recolección de datos: Alimentar el modelo con información real.
  4. Implementación en software: Programar el modelo.

Etapas de un Proceso de Simulación (Cont.)

  1. Validación: ¿El modelo se comporta como el sistema real?
  2. Experimentación: Ejecutar la simulación con diferentes escenarios.
  3. Interpretación: Analizar los resultados y sacar conclusiones.
  4. Documentación: Crear informes y manuales.

Tipos de Modelos Matemáticos

  • Ecuaciones Diferenciales: El método clásico. Complejo para sistemas no lineales o multivariables.
  • Función de Transferencia: Informa sobre la estabilidad, pero oculta la arquitectura interna.
  • Variables de Estado (Espacio de Estados): El enfoque moderno. Versátil, detallado y perfecto para la simulación computacional.

Análisis por Variables de Estado

Preguntas Clave de esta Sección:

  • ¿Qué representa un punto en el "espacio de estados"?
  • ¿Cómo se determina el orden de un sistema?
  • ¿Qué son las ecuaciones de estado y qué representa cada matriz?

Conceptos Fundamentales

  • Estado: Descripción completa de la condición dinámica de un sistema en un instante. Es la "foto" del sistema.
  • Variables de Estado: El conjunto mínimo de parámetros que definen el estado. Se asocian a elementos que almacenan energía.
  • Orden del Sistema (n): Número de variables de estado.

Elementos Almacenadores de Energía

Sistemas Eléctricos

La energía se almacena en campos magnéticos y eléctricos.

  • Inductores (L) → Variable: Corriente ($i_L$)
  • Capacitores (C) → Variable: Tensión ($v_C$)

Sistemas Mecánicos

La energía se almacena como energía cinética y potencial.

  • Masas (m) → Variable: Velocidad ($v$)
  • Resortes (k) → Variable: Posición ($x$)

Espacio de Estados

  • Espacio de Estados: Un espacio n-dimensional donde cada eje es una variable de estado.
  • Vector de Estado (x): Un vector que une el origen con el punto que representa el estado actual del sistema.
  • Trayectoria de Estado: La curva que dibuja el vector de estado a lo largo del tiempo.

Las Ecuaciones de Estado

Estas dos ecuaciones son el corazón del modelo de estado. Definen toda la dinámica del sistema.

Ecuación de Estado: $\dot{\mathbf{x}}(t) = \mathbf{A} \mathbf{x}(t) + \mathbf{B} \mathbf{u}(t)$

Describe cómo evoluciona el estado del sistema en el tiempo.

Ecuación de Salida: $\mathbf{y}(t) = \mathbf{C} \mathbf{x}(t) + \mathbf{D} \mathbf{u}(t)$

Describe cómo se calcula la salida que observamos.

Las Matrices del Sistema

  • $\mathbf{x}(t)$: Vector de estado (n x 1)
  • $\mathbf{u}(t)$: Vector de entrada (m x 1)
  • $\mathbf{y}(t)$: Vector de salida (p x 1)
  • $\mathbf{A}$: Matriz de estado (n x n) - La dinámica interna.
  • $\mathbf{B}$: Matriz de entrada (n x m) - Cómo afecta la entrada al estado.
  • $\mathbf{C}$: Matriz de salida (p x n) - Cómo se mide la salida desde el estado.
  • $\mathbf{D}$: Matriz de transmisión directa (p x m) - Conexión directa entrada-salida (a menudo es cero).

Ejemplo 1: Circuito RLC en Serie

Vamos a obtener el modelo de estado paso a paso.

Diagrama: Circuito RLC en Serie

(El alumno puede buscar una imagen de referencia para un circuito RLC en serie)

Ejemplo 1: Circuito RLC en Serie

Paso 1: Identificar Variables de Estado

Los elementos que almacenan energía son L y C.

  • $x_1(t) = i_L(t)$ (Corriente en el inductor)
  • $x_2(t) = v_C(t)$ (Tensión en el capacitor)

El orden del sistema es 2.

Ejemplo 1: Circuito RLC en Serie

Paso 2: Ecuaciones del Sistema (Leyes de Kirchhoff)

Ley de Voltajes (malla):

$v_i(t) = R i(t) + L \frac{di(t)}{dt} + v_C(t)$

Corriente en el capacitor:

$i(t) = C \frac{dv_C(t)}{dt}$

Ejemplo 1: Circuito RLC en Serie

Paso 3: Formular Ecuaciones de Estado

Despejamos las derivadas de las variables de estado:

$\dot{x}_1 = \frac{di}{dt} = -\frac{R}{L}x_1 - \frac{1}{L}x_2 + \frac{1}{L}v_i(t)$

$\dot{x}_2 = \frac{dv_C}{dt} = \frac{1}{C}x_1$

Ejemplo 1: Circuito RLC en Serie

Paso 4: Representación Matricial

Ecuación de Estado:

$\begin{Bmatrix} \dot{x}_1 \\ \dot{x}_2 \end{Bmatrix} = \begin{bmatrix} -\frac{R}{L} & -\frac{1}{L} \\ \frac{1}{C} & 0 \end{bmatrix} \begin{Bmatrix} x_1 \\ x_2 \end{Bmatrix} + \begin{Bmatrix} \frac{1}{L} \\ 0 \end{Bmatrix} v_i(t)$

Ecuación de Salida (si $y = v_C$):

$y(t) = \begin{bmatrix} 0 & 1 \end{bmatrix} \begin{Bmatrix} x_1 \\ x_2 \end{Bmatrix}$

Con este modelo, podemos simular el comportamiento del circuito para cualquier entrada $v_i(t)$ y condiciones iniciales.

Ejemplo 2: Sistema Masa-Resorte-Amortiguador

Diagrama: Sistema Masa-Resorte-Amortiguador

(El alumno puede buscar una imagen de referencia)

Ejemplo 2: Sistema Masa-Resorte-Amortiguador

Paso 1: Identificar Variables de Estado

Los elementos que almacenan energía son la masa (energía cinética) y el resorte (energía potencial).

  • $x_1(t) = y(t)$ (Posición de la masa)
  • $x_2(t) = \dot{y}(t)$ (Velocidad de la masa)

Ejemplo 2: Sistema Masa-Resorte-Amortiguador

Paso 2: Ecuación del Sistema (2da Ley de Newton)

$\sum F = ma \implies u(t) - k y(t) - B \dot{y}(t) = m \ddot{y}(t)$

Despejando la aceleración:

$\ddot{y}(t) = -\frac{k}{m} y(t) - \frac{B}{m} \dot{y}(t) + \frac{1}{m} u(t)$

Ejemplo 2: Sistema Masa-Resorte-Amortiguador

Paso 3: Formular Ecuaciones de Estado

A partir de la definición de las variables:

$\dot{x}_1 = \dot{y} = x_2$

$\dot{x}_2 = \ddot{y} = -\frac{k}{m} x_1 - \frac{B}{m} x_2 + \frac{1}{m} u(t)$

Ejemplo 2: Sistema Masa-Resorte-Amortiguador

Paso 4: Representación Matricial

Ecuación de Estado:

$\begin{Bmatrix} \dot{x}_1 \\ \dot{x}_2 \end{Bmatrix} = \begin{bmatrix} 0 & 1 \\ -\frac{k}{m} & -\frac{B}{m} \end{bmatrix} \begin{Bmatrix} x_1 \\ x_2 \end{Bmatrix} + \begin{Bmatrix} 0 \\ \frac{1}{m} \end{Bmatrix} u(t)$

Ecuación de Salida (si $y$ es la posición):

$y(t) = \begin{bmatrix} 1 & 0 \end{bmatrix} \begin{Bmatrix} x_1 \\ x_2 \end{Bmatrix}$

Este modelo nos permite predecir la posición y velocidad de la masa ante cualquier fuerza $u(t)$.

Resolución del Modelo de Estado

Resolver la ecuación de estado significa encontrar el vector $\mathbf{x}(t)$. Usaremos la Transformada de Laplace.

$\dot{\mathbf{x}}(t) = \mathbf{A} \mathbf{x}(t) + \mathbf{B} \mathbf{u}(t)$

Aplicando Laplace:

$s\mathbf{X}(s) - \mathbf{x}(0) = \mathbf{A}\mathbf{X}(s) + \mathbf{B}\mathbf{U}(s)$

Despejando X(s)

$s\mathbf{X}(s) - \mathbf{A}\mathbf{X}(s) = \mathbf{x}(0) + \mathbf{B}\mathbf{U}(s)$

$(s\mathbf{I} - \mathbf{A})\mathbf{X}(s) = \mathbf{x}(0) + \mathbf{B}\mathbf{U}(s)$

La solución en el dominio de Laplace es:

$\mathbf{X}(s) = (s\mathbf{I} - \mathbf{A})^{-1}\mathbf{x}(0) + (s\mathbf{I} - \mathbf{A})^{-1}\mathbf{B}\mathbf{U}(s)$

Donde $(s\mathbf{I} - \mathbf{A})$ es la Matriz Característica del sistema.

La Matriz de Transición de Estado $\Phi(t)$

La solución en el dominio del tiempo tiene dos partes:

  • Respuesta a las condiciones iniciales (entrada cero).
  • Respuesta a la entrada (condiciones iniciales cero).

$\mathbf{x}(t) = \mathcal{L}^{-1}\{(s\mathbf{I} - \mathbf{A})^{-1}\}\mathbf{x}(0) + \mathcal{L}^{-1}\{(s\mathbf{I} - \mathbf{A})^{-1}\mathbf{B}\mathbf{U}(s)\}$

Definimos la Matriz de Transición de Estado como:

$\Phi(t) = \mathcal{L}^{-1}\{(s\mathbf{I} - \mathbf{A})^{-1}\} = e^{\mathbf{A}t}$

Esta matriz "hace la transición" del estado desde $t=0$ hasta un tiempo $t$.

Solución Final en el Tiempo

La solución completa es:

$\mathbf{x}(t) = \Phi(t)\mathbf{x}(0) + \int_{0}^{t} \Phi(t-\tau)\mathbf{B}\mathbf{u}(\tau)d\tau$

Esta es la integral de convolución. Nos dice que el estado en cualquier momento es la suma de la evolución natural del sistema (respuesta libre) y la respuesta forzada por la entrada.

Interconexión de Sistemas

Los sistemas complejos se construyen a partir de subsistemas más simples. Analizaremos cómo modelar estas conexiones.

Conexión en Serie (Cascada)

La salida de un sistema es la entrada del siguiente: $u_2(t) = y_1(t)$

Diagrama: Conexión en Serie

u1(t) → [ S1 ] → y1(t)=u2(t) → [ S2 ] → y2(t)

La función de transferencia equivalente es el producto:

$H(s) = H_1(s)H_2(s)$

El modelo de estado se construye con un vector de estado aumentado: $\mathbf{x} = \begin{bmatrix} \mathbf{x}_1 \\ \mathbf{x}_2 \end{bmatrix}$

Conexión Retroalimentada (Lazo Cerrado)

La salida se "realimenta" y se compara con una referencia.

Diagrama: Conexión Retroalimentada

r(t) → (Σ) → [ S1 ] → y(t)

          ↑--- [ S2 ] ←┘

La función de transferencia equivalente es:

$H(s) = \frac{H_1(s)}{1 + H_1(s)H_2(s)}$

Pon a Prueba tu Comprensión

Pregunta 1:

Si un sistema mecánico tiene 3 masas y 2 resortes, ¿cuál es el orden mínimo del sistema y por qué?

Respuesta 1

El orden del sistema es 5.

Razón: El orden del sistema es igual al número de elementos almacenadores de energía independientes. Cada masa almacena energía cinética (requiere una variable de estado: velocidad) y cada resorte almacena energía potencial (requiere una variable de estado: posición/elongación). Por lo tanto, necesitamos 3 variables para las velocidades de las masas y 2 para las posiciones de los resortes, sumando un total de 5 variables de estado.

Pon a Prueba tu Comprensión

Pregunta 2:

En la ecuación $\dot{x} = Ax + Bu$, ¿qué representa físicamente la matriz A y qué representa la matriz B?

Respuesta 2

  • La Matriz A (Matriz de Estado) representa la dinámica interna del sistema. Describe cómo las variables de estado interactúan entre sí sin ninguna influencia externa. Sus valores dependen de los parámetros físicos del sistema (R, L, C, k, m, B, etc.).
  • La Matriz B (Matriz de Entrada) representa cómo la entrada externa $u(t)$ afecta a cada una de las variables de estado. Actúa como un "distribuidor" de la energía o fuerza de entrada al sistema.

Resumen de la Unidad 1

  • Los modelos y la simulación son herramientas clave para analizar sistemas de forma segura y económica.
  • La representación por variables de estado es el método moderno y más versátil.
  • Las ecuaciones de estado ($\dot{x}=Ax+Bu, y=Cx+Du$) describen matemáticamente cualquier sistema lineal.
  • Podemos obtener el modelo de estado a partir de las leyes físicas del sistema (Newton, Kirchhoff).
  • La solución se encuentra usando la Transformada de Laplace y la Matriz de Transición de Estado $\Phi(t)$.
  • Podemos modelar sistemas complejos interconectando subsistemas.
