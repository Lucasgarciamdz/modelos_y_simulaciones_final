Objetivo: Comprender los fundamentos teóricos que sustentan el modelado y la simulación de sistemas.

¿Qué es un Modelo?

Un modelo es una descripción analítica y abstracta de un sistema, proceso o circuito, formulada para estudiar su comportamiento bajo diferentes condiciones de operación.

Concepto Moderno

También conocido como gemelo digital (digital twin)

  • Réplica virtual
  • Actualización en tiempo real
  • Predicción de comportamiento

¿Por qué Modelar?

Cuando la experimentación directa es inconveniente por:

  • Seguridad
  • Costo
  • Practicidad

Ejemplos de Aplicación del Modelado

Ensayos Destructivos

Punto de ruptura de vigas sin comprometer la estructura real

Investigaciones Biológicas

Respuesta del corazón a temperaturas extremas

Aplicaciones Militares

Poder destructivo de explosivos sobre estructuras civiles

Análisis de Radioactividad

Efectos de radiación sobre organismos

Principio Fundamental: Estudiar el sistema sin comprometerlo

¿Qué es la Simulación?

La simulación es el proceso de diseñar, desarrollar y experimentar con un modelo programable de un sistema para analizar su comportamiento bajo distintas condiciones.

Visión Específica:

Técnica numérica que utiliza computadoras para conducir experimentos sobre modelos matemáticos y lógicos que describen un sistema.

Ventaja clave: Permite recrear diferentes entornos y explorar escenarios hipotéticos sin interrumpir las operaciones del mundo real.

Evolución del Modelado

Método Tradicional

  • Ecuaciones diferenciales de equilibrio
  • Análisis matemático complejo
  • Limitado a sistemas lineales
  • Difícil para sistemas multivariables

Método Moderno

  • Computadoras digitales
  • Modelo algorítmico
  • Procesamiento a alta velocidad
  • Mayor exactitud
  • Diversificación de escenarios

Impacto: Las computadoras han revolucionado la capacidad de simulación y análisis de sistemas complejos.

Etapas de un Proceso de Simulación

  1. Definición del sistema y formulación del problema
  2. Formulación del modelo matemático
  3. Recolección de datos históricos y experimentales
  4. Implementación en software especializado
  5. Validación del modelo
  6. Experimentación con diferentes escenarios
  7. Interpretación de resultados
  8. Documentación técnica

Variables de Estado

¿Qué responderemos en esta sección?

  • ¿Qué es el estado de un sistema?
  • ¿Cómo se definen las variables de estado?
  • ¿Qué información proporciona el espacio de estados?

Objetivo: Dominar los conceptos fundamentales del análisis por variables de estado.

Conceptos Fundamentales

Estado

Descripción completa de la condición dinámica de un sistema en un instante específico (t₀). Conocer el estado en t₀ y las entradas para t ≥ t₀ es suficiente para determinar el comportamiento futuro.

Variables de Estado

Conjunto mínimo de variables que definen completamente el estado del sistema. Se asocian a elementos almacenadores de energía.

Conceptos Adicionales

Espacio de Estados

Espacio n-dimensional donde cada eje corresponde a una variable de estado. Un punto define el estado del sistema.

Vector de Estado

Vector que une el origen con el punto que representa el estado del sistema.

Trayectoria de Estado

Secuencia de puntos que representa la evolución del sistema a lo largo del tiempo.

Orden del Sistema

Número de elementos almacenadores de energía = número de variables de estado (n).

Elementos Almacenadores de Energía

Sistemas Eléctricos

  • Inductores (L): Campo magnético
  • Capacitores (C): Carga eléctrica
  • Variables de estado:
    • Corriente en inductor (i_L)
    • Voltaje en capacitor (v_C)

Resistencia (R) = elemento disipativo

Sistemas Mecánicos

  • Masa (m): Energía cinética
  • Resorte (k): Energía potencial
  • Variables de estado:
    • Posición (x)
    • Velocidad (v)

Amortiguador (B) = elemento disipativo

Ecuaciones de Estado - Forma General

Ecuación de Estado

$$\dot{\mathbf{x}}(t) = \mathbf{A} \mathbf{x}(t) + \mathbf{B} \mathbf{u}(t)$$

Ecuación de Salida

$$\mathbf{y}(t) = \mathbf{C} \mathbf{x}(t) + \mathbf{D} \mathbf{u}(t)$$

  • x(t): vector de estado
  • ẋ(t): derivadas de variables de estado
  • u(t): vector de entrada
  • y(t): vector de salida
  • A: matriz de estado
  • B: matriz de entrada
  • C: matriz de salida
  • D: matriz de transmisión directa

Ventajas de la Representación por Variables de Estado

1. Análisis Simplificado

Se pueden extraer conclusiones examinando las propiedades de las matrices.

2. Aplicabilidad Universal

Válido para sistemas de orden superior, no lineales y MIMO.

3. Implementación Computacional

Facilita el desarrollo de algoritmos para resolución numérica y simulación.

PASO 1: Definir Variables de Estado

x₁(t) = i(t) → Corriente en el inductor

x₂(t) = v₀(t) → Voltaje en el capacitor

Ejemplo: Circuito RLC en Serie

PASO 2: Aplicar Leyes de Kirchhoff

Ley de Voltajes (LVK):

$$v_i(t) = L \frac{di(t)}{dt} + R i(t) + v_o(t)$$

Corriente en el capacitor:

$$i(t) = C \frac{dv_o(t)}{dt}$$

Aplicamos las leyes fundamentales de circuitos...

Ejemplo: Circuito RLC en Serie

PASO 3: Reordenar las Ecuaciones

$$\frac{dx_1}{dt} = \dot{x}_1 = -\frac{R}{L}x_1 - \frac{1}{L}x_2 + \frac{1}{L}v_i$$

$$\frac{dx_2}{dt} = \dot{x}_2 = \frac{1}{C}x_1$$

Aislamos las derivadas de las variables de estado...

Ejemplo: Circuito RLC en Serie

PASO 4: Representación Matricial

$$\begin{bmatrix} \dot{x}_1 \\ \dot{x}_2 \end{bmatrix} = \begin{bmatrix} -\frac{R}{L} & -\frac{1}{L} \\ \frac{1}{C} & 0 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} + \begin{bmatrix} \frac{1}{L} \\ 0 \end{bmatrix} v_i(t)$$

$$y(t) = \begin{bmatrix} 0 & 1 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}$$

¡Ecuaciones de estado completas!

Sistema mecánico de 2º orden

Variables de Estado

x₁(t) = y(t) → Posición

x₂(t) = ẏ(t) → Velocidad

Ecuación de Equilibrio

$$u(t) - ky(t) - B\frac{dy}{dt} = m\frac{d^2y}{dt^2}$$

Sistema Mecánico - Ecuaciones de Estado

$$\begin{bmatrix} \dot{x}_1 \\ \dot{x}_2 \end{bmatrix} = \begin{bmatrix} 0 & 1 \\ -\frac{k}{m} & -\frac{B}{m} \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} + \begin{bmatrix} 0 \\ \frac{1}{m} \end{bmatrix} u(t)$$

$$y(t) = \begin{bmatrix} 1 & 0 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}$$

Observación: La estructura de las matrices refleja directamente la física del sistema. La matriz A contiene la dinámica interna, B la influencia de las entradas, y C define qué medimos como salida.

Puntos de Equilibrio

Definición

Un punto de equilibrio es un estado donde las derivadas de todas las variables de estado son cero:

$$\dot{\mathbf{x}} = \mathbf{0}$$

Características

  • Condición estacionaria
  • Sin cambios en el tiempo
  • Pueden ser estables o inestables

Importancia

  • Puntos de operación
  • Análisis de estabilidad
  • Diseño de controladores

Diferencia Fundamental

Modelo: Es la representación abstracta y matemática de un sistema. Es estático, como un "plano" o "esquema" del sistema.

Simulación: Es el proceso dinámico de ejecutar el modelo para observar su comportamiento a lo largo del tiempo.

¿Por qué la representación por variables de estado es preferible a las ecuaciones diferenciales tradicionales?

Menciona al menos tres ventajas específicas.

Considera sistemas complejos y la implementación computacional...

Respuesta 2

Ventajas de Variables de Estado

  1. Aplicabilidad Universal: Funciona para sistemas lineales y no lineales, SISO y MIMO, variantes e invariantes en el tiempo.
  2. Implementación Computacional: Las ecuaciones matriciales son ideales para procesamiento digital y algoritmos numéricos.
  3. Información Detallada: Proporciona conocimiento completo del estado interno del sistema, no solo la relación entrada-salida.

Pregunta 3

Para un motor DC con carga mecánica, ¿cuáles serían las variables de estado apropiadas y por qué?

Considera que el motor tiene inercia rotacional J, fricción viscosa B, y está conectado a una fuente de voltaje.

Piensa en los elementos que almacenan energía...

Respuesta 3

Variables de Estado para Motor DC

Variables Propuestas

x₁(t) = i(t) → Corriente en la armadura

x₂(t) = ω(t) → Velocidad angular

Justificación

Corriente: Representa la energía almacenada en la inductancia de la armadura

Velocidad angular: Representa la energía cinética rotacional almacenada en la inercia J

Principio: Las variables de estado se asocian con elementos almacenadores de energía: inductancia eléctrica (L) e inercia mecánica (J).

Resumen de la Unidad 1

Puntos Clave Fundamentales

  1. Modelado es Esencial: Permite estudiar sistemas sin riesgos, costos o limitaciones prácticas. El concepto de "gemelo digital" es la evolución moderna.
  2. Simulación vs Modelo: El modelo es la representación estática; la simulación es su ejecución dinámica para analizar comportamiento temporal.
  3. Variables de Estado: Método preferencial por su versatilidad, detalle informativo y compatibilidad computacional. Se basa en elementos almacenadores de energía.
  4. Aplicación Práctica: Las ecuaciones de estado proporcionan un framework sistemático para analizar cualquier sistema dinámico, desde circuitos eléctricos hasta sistemas mecánicos complejos.
